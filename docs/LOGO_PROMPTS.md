Boutique beauté & accessoires capillaires pour femmes réunionnaises.
Esprit : tropical, féminin, accessible, moderne. Pas luxe, pas cheap.
Couleurs : rose hibiscus (#C8435A), vert tropical (#2D6A4F), accent doré (#F4A261), fond crème (#FFF8F3).

```
with clean lines. The flower should be simplified to 4-5 petals
in a flat design style. Color: deep rose pink (#C8435A) on a
warm cream background (#FFF8F3). Below the flower, the text
"KRÉOL GLOW" in elegant serif font (Playfair Display style),

No gradients, no shadows, flat design.
Style: modern minimalist, Aesop meets Fenty Beauty.
Format: square, suitable for Instagram profile picture.
White background version.
```

## 📐 Spécifications techniques pour le designer/IA

- Format de livraison souhaité : SVG + PNG (transparence) + PNG fond blanc
- Tailles : 512x512 (carré), 1200x400 (horizontal), 32x32 (favicon)
- Usage : web, réseaux sociaux, packaging kraft, stickers
- Contrainte : lisible en noir et blanc (pour tampons/tampons encreurs)

## 🛠️ Outils gratuits recommandés par ordre de qualité

1. **Ideogram.ai** → meilleur pour le texte dans les logos (gratuit, 25 générations/jour)
2. **Gemini 2.0** → bonne compréhension du brief, gratuit
3. **Canva Magic Design** → si tu veux un logo éditabe rapidement, moins original
4. **Looka.com** → IA logo maker spécialisé, 20$/one-time pour les fichiers SVG
5. **DALL-E 3 via ChatGPT** → bon résultats mais texte parfois déformé
